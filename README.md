# SmartShelfX – AI-Based Inventory System  
Infosys Springboard Virtual Internship 6.0

SmartShelfX is an AI-based inventory management system focused on demand forecasting and restock analysis for retail environments. The project demonstrates intelligent inventory planning techniques as part of the Infosys Springboard Virtual Internship 6.0.

## Tech Stack
- Frontend: React.js  
- Backend: Java Spring Boot  
- Database: MySQL  
- Tools: Git, GitHub
